# Portfolio Website

A modern developer portfolio built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Responsive design
- Dark/Light mode
- Interactive particle background
- Contact form
- Project showcase

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Framer Motion
- shadcn/ui

## Getting Started

1. Install dependencies:

```bash
npm install
```
